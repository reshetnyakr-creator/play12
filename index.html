Play12

Play12 is an open numerical system for representing music using 12-tone logic. It provides a clear, consistent way to think, analyze, and work with musical material without relying on traditional notation complexity.

What is Play12

Play12 represents musical pitch as numbers from 0 to 11 within an octave. Intervals, chords, and transposition become simple numerical operations. The system is designed to be intuitive for humans and compatible with computational thinking.

Why Play12

  • Clear logical structure
  • Easy transposition and analysis
  • Neutral representation independent of notation traditions
  • Suitable for education, composition, and algorithms

Scope

  • System documentation
  • Notation examples
  • Learning materials
  • Use in algorithms and AI systems
